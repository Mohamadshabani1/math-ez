What is the linear equation?

A linear equation is an equation for a straight line consisting of only one variable

The linear equation:

y=mx+b

The linear equation has only one solution because it is a first degree equations so the solution for it is just to find the value of the variable X

we can find the answer with just 2 steps!

First step:

y-b=mx+b-b

Second step:

x=(y-b)/m
